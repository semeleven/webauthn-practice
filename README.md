# Как использовать
```
npm install
npm run build
HOST_NAME=... npm run start:server
```
Сейчас вы видите последнюю демку. Остальные демки в предыдущих комитах, вы можете перейти к ним, например `git checkout 1-password-login`
`HOST_NAME` это указание публичного адреса. Для его получения можете использовать [ngrok](https://ngrok.com)
